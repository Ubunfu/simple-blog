SVG Graphics

October 11, 2017 @ 10:18

SVG graphics can be used as drop-in replacements for most non-photographic images such as logos or graphics. These should be used whenever possible because they are based on calculated vectors, so they scale infinitely with no loss of definition. You don't have to have multiple versions of your images to support different UIs.

SVGs are nothing more than XML, and are rendered by the browser. You can pop them open and read/modify the contents, or even paste the SVG code directly in an HTML document. If you do it this way, it's called an "inline SVG". Inline SVGs are nice because you can style them using CSS like you would any other DOM element in your site.

Video

October 5, 2017 @ 20:56

Use these attributes in the video tag:

  • controls (boolean) : Show on-screen video controls.
  • autoplay (boolean) : Video plays automatically on page-load.
  • poster : Path to a placeholder image before the video plays.
  • preload (boolean) : If set, browser will try to automatically start downloading the video when the page loads. Does not work on mobile browsers b/c mobile data plans. :(
  • tabindex : Zero-based integer index. Allows user to tab through a page full of stuff and use the keyboard to control the video.
  • loop (boolean) : Auto-loops the video. Forever...ever...ever...
  • muted (boolean) : Video will not have any sound.

Check out this cool site on the NY Times for some HTML 5 video inspiration.

Video For Everyone:

Video For Everyone is a web service which will generate HTML 5 snipped that includes a flash fallback. The link to the supporting flash runtime files are hosted links, so for mission critical sites it would be wise to host them locally.

Miro Video Converter:

Miro Video Converter is a tool that will convert video files between nearly any formats. This comes in handy when designing a compatibility solution for HTML5 videos.
