This is the main content of the blog post! It is an article tag. Herrr's the link: HTML5 Article

This is a really interesting tag. Apparently, its symantic meaning is: "a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable (e.g., in syndication). Examples include: a forum post, a magazine or newspaper article, or a blog entry."

Sounds perfect, right?...

In the upper right is a time element. Herr is that link too: HTML5 Time

This time element allows the browser to understand that the value inside of the "datetime" parameter is the true date and time associated with this article. I'm still not quite sure what the use of that is.It also seems pretty manual.. maybe there is a way to use it dynamically or something.

The "pubdate" parameter is a boolean that tells the browser even further that this is the date at which the article was actually published. Not sure what I can do with that yet.
